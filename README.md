Hotel Booking Demand – Prediction Project

# Project Introduction
Project ini berfokus pada pembuatan model machine learning untuk memprediksi apakah sebuah pemesanan hotel akan dibatalkan atau tidak. 
Analisis dilakukan menggunakan data historis pemesanan, lalu dilanjutkan dengan pembersihan data, eksplorasi, dan pengembangan model.

Project ini bertujuan untuk membantu pihak hotel mengurangi kerugian akibat pembatalan mendadak serta meningkatkan akurasi perencanaan okupansi.

Proses pengerjaan mencakup:
1. Exploratory Data Analysis (EDA)
2. Data preprocessing
3. Feature selection
4. Model training & evaluation
5. Deployment-ready model (Pickle file)

# Project Structure
- **Capstone_Project_Hotel_Booking_Demand.ipynb** → Dokumentasi analisis lengkap dan Proses pembuatan model.
- **model_booking.pkl** → Model final yang sudah disimpan menggunakan Pickle.
- **README.md** → Penjelasan singkat project.
