# Project Introduction
Sebuah hotel yang berlokasi di Portugal menghadapi tingginya jumlah pembatalan pemesanan kamar. Banyak pelanggan yang melakukan booking namun membatalkannya sebelum hari kedatangan. 
Kondisi ini menyebabkan hotel kehilangan potensi pendapatan, kesulitan dalam memaksimalkan tingkat hunian kamar, serta mengganggu perencanaan operasional seperti alokasi kamar dan penjadwalan staf.
Sehingga untuk mengatasi masalah tersebut, manajemen hotel ingin mengetahui dan memprediksi lebih awal apakah sebuah pelanggan yang telah melakukan booking berpotensi dibatalkan atau tidak. 
Informasi terkait negara asal, tipe deposit, tipe kamar, kebutuhan tempat parkir mobil dan permintaan spesial ada di tangan dari pelanggan hotel.

# Problem Statement
Tingginya pembatalan reservasi menimbulkan kerugian karena kamar yang dibatalkan sering kali tidak dapat dijual kembali dalam waktu singkat. 
Jika hotel memperlakukan semua reservasi secara sama tanpa mempertimbangkan risiko pembatalan, maka proses perencanaan okupansi, strategi overbooking, dan pengelolaan sumber daya menjadi tidak efisien.
Ketidakpastian ini mengganggu alokasi tenaga kerja, persiapan kamar, serta perencanaan kapasitas. Akibatnya, banyak usaha operasional berpotensi terbuang apabila tamu tidak datang.

# Tujuan
Tujuan utama proyek ini adalah membangun model prediktif untuk mengidentifikasi kemungkinan pembatalan reservasi. Dengan model ini, manajemen hotel dapat:
* Meminimalkan kerugian finansial
* Meningkatkan akurasi perencanaan okupansi
* Mendukung strategi bisnis seperti penetapan harga dan overbooking
* Memahami faktor-faktor yang memengaruhi pembatalan tamu

# Pendekatan Analitik
Pendekatan analitik yang digunakan mencakup:
* Pembersihan dan preprocessing data
* Pemilihan fitur yang relevan
* Encoding variabel kategorikal
* Pembangunan model klasifikasi untuk memprediksi pembatalan
* Evaluasi menggunakan metrik **Recall** dan **AUC-ROC** untuk memastikan model mampu mengenali reservasi berisiko tinggi
* Analisis faktor utama yang mendorong pembatalan untuk dasar rekomendasi bisnis

# Stakeholders
Model dan hasil analisis ini ditujukan untuk mendukung pengambilan keputusan oleh:
* **Manajer Operasional:** memastikan kelancaran operasional hotel
* **Manajer Pemasaran:** menyusun strategi untuk menarik tamu
* **Manajer Pendapatan:** mengoptimalkan strategi harga dan ketersediaan kamar
* **Pemilik Hotel:** memantau performa bisnis dan menetapkan arah strategi jangka panjang
